What is MetoCities

MetoCities thrives to be as realistic as possible in a game server. A lot of real concepts are precisely transferred to the server so that you, the player, can have the most realistic experience of any servers. Our team works around the clock developing, upgrading, and managing the server so that it may run smoothly. In game aspects include a currency, vehicles, air travel, jobs, property market, states, cities, and more.
